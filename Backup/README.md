# Backup

Este repositorio contiene scripts de automatización para diferentes tareas.

## Scripts

- autoBackup.sh: Script para hacer backup de directorios a tar.gz

## Uso

```bash
./autoBackup.sh
```

## Explicación

El script realiza las siguientes acciones:

1. Recoge los directorios a respaldar
2. Muestra un resumen
3. Directorio de destino
4. Nombre del archivo
5. Nivel de compresión
6. Confirmación final
7. Crear el backup
8. Verificación y resultado
